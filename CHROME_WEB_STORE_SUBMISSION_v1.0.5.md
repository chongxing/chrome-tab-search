#### Detailed Description
```
🔍 Tired of hunting through dozens of tabs across multiple windows?

Tab Search helps you instantly find and jump to any tab with just a few keystrokes!

✨ KEY FEATURES:

🔎 Cross-Window Search
   - Search across ALL Chrome windows simultaneously
   - Not limited to the current window like Chrome's built-in tab search

⚡ Instant Results
   - Real-time filtering as you type
   - Search by tab title or URL

🕐 Sort by Last Access Time (NEW in v1.0.5!)
   - Switch to "Time" view to see recently accessed tabs first
   - Perfect for: "I just saw that page, where was it?"
   - Automatically tracks when you switch to or refresh a tab

🎯 One-Click Jump
   - Select a tab and instantly focus its window
   - No more manual window switching!

⌨️ Keyboard-First Design
   - Ctrl+Shift+T (Mac: ⌘+Shift+T) to open
   - Arrow keys to navigate, Enter to jump, Esc to close
   - Never touch your mouse

🪟 Smart Window Grouping
   - Group tabs by window in "Window" mode
   - Current window displayed first
   - Shows window name and tab count

🔊 Tab Status Indicators
   - See which tab is currently active
   - Identify tabs playing audio

🌐 WHO IS THIS FOR?

• Developers working with multiple documentation pages
• Researchers managing numerous reference tabs
• Analysts comparing data across windows
• Anyone who keeps 20+ tabs open!

🔒 PRIVACY & SECURITY

• 100% local operation - no data leaves your browser
• Only reads tab metadata (title, URL) - never page content
• No ads, no tracking, no analytics
• Minimal permissions: tabs, windows, storage
• Open source: github.com/chongxing/chrome-tab-search

⌨️ CUSTOMIZE SHORTCUTS

Visit chrome://extensions/shortcuts to change the keyboard shortcut.

---

💡 PRO TIP: Press "2" in the popup to quickly switch to time-based sorting!

📝 Version History:
• v1.0.5 - Added last access time tracking and sorting
• v1.0.4 - Chrome Web Store release with time-based sorting
• v1.0.3 - Fixed pre-existing tab sorting
• v1.0.2 - Added time-based sorting feature
• v1.0.1 - Initial release
```
