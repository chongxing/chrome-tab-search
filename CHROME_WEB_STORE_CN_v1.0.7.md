# Chrome Web Store 上传资料 v1.0.7（中文）

## 版本信息
- **版本号**: 1.0.7
- **发布日期**: 2026-03-25
- **更新类型**: 功能更新

## 更新内容

### ✨ 新功能：搜索状态保持
- **记住搜索关键字**：关闭弹窗后重新打开，搜索框内容自动恢复
- **记住选中位置**：上次选中的 tab 索引会被保存
- **无缝浏览体验**：跳转到某个 tab 后发现不对，重新打开 Tab Search 可从之前的位置继续浏览
- **智能清除**：按 `Esc` 关闭时清除选中位置，但保留搜索关键字

### 🎯 改进
- 提升多标签页浏览的用户体验
- 减少重复输入相同关键字的操作
- 在搜索结果中迭代浏览时更快速

## Store Listing 信息

Tab Search - 跨窗口标签页搜索与置顶管理

### 简短描述（最多132字符）
跨所有窗口快速搜索、置顶和跳转到任意标签页。持久化搜索+时间排序，高效管理海量标签。

### 详细描述
🔍 **Tab Search - 跨窗口标签页管理器** 是一款强大的标签页管理工具，帮助你在所有浏览器窗口中瞬间找到并切换到任意标签页。

**核心功能：**

🔍 **全新：搜索状态保持**
- 重新打开弹窗时自动恢复上次的搜索关键字
- 保存上次选中的位置
- 从之前的位置继续搜索，无需重新开始
- 告别重复输入相同关键字

📌 **置顶重要标签**
- 将常用标签置顶，始终显示在列表顶部
- 点击置顶按钮或按 `P` 键快速切换
- 置顶状态跨浏览器会话保持
- 在两种排序模式下都有效

🪟 **两种排序模式**
- **窗口模式**：按窗口分组显示，当前窗口优先
- **时间模式**：按最后访问时间排序，最近访问在前

⚡ **极速体验**
- 快捷键：Ctrl+Shift+T（Mac: Cmd+Shift+T）
- 方向键导航，回车跳转
- 即时搜索，零延迟

🎨 **简洁现代界面**
- 美观、无干扰的界面设计
- 清晰的窗口分组层级
- 当前标签和播放声音标签标识

**隐私保护：**
- 无数据收集或追踪
- 所有数据保存在本地
- 仅访问你已可见的标签信息

**所需权限：**
- `tabs`：搜索和切换标签
- `windows`：查找所有窗口中的标签
- `storage`：保存置顶状态、偏好设置和搜索状态

适合需要同时管理多个窗口数十个标签的重度用户！

### ✨ 新功能
- **持久化搜索关键字**：搜索关键字在关闭弹窗后重新打开时自动恢复

- `filterTabs()` 改为 async 并保存查询到 chrome.storage.local
- `switchToTab()` 跳转前保存选中索引
- Escape 键处理程序更新为清除选中索引
